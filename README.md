## Launch 


```bash
cargo run -- --help
Options:
  -t, --time       Calculate execution time
  -d, --day <DAY>  Days to evaluate [default: 1]
  -h, --help       Print help
  -V, --version    Print version

    # ex
    cargo run -- -d 1 -t
```
